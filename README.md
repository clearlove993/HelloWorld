# HelloWorld
## 项目的简历及其相关的用

这是一个可以实现命令行文本计数统计程序。

输入的命令模式为：wc.exe [参数] [文件名]

参数说明：

* 统计字符数    -c

* 统计单词数    -w

* 统计句子数    -s

* 统计行数        -l

* 统计空行        -b

* 统计注释行    -f

举个栗子：

wc.exe -c file.txt           统计字符数

wc.exe -w file.txt           统计单词数
